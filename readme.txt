خطوات تنظيف البيانات:

إزالة التكرار
اتأكدت إن مفيش صفوف مكررة وحذفت الدوبليكيت.

توحيد البيانات
ظبطت الفورمات والقيم علشان تبقى موحدة ومفيهاش أخطاء.

التعامل مع القيم الفارغة
راجعت الـ Null values وشفت أتعامل معاها إزاي.

حذف البيانات غير المهمة
شلت الأعمدة والصفوف اللي مش محتاجها في التحليل.

Data Cleaning Steps:

Removed Duplicates
Checked the dataset and removed duplicate rows.

Standardized the Data
Fixed formatting issues and made the data consistent.

Handled Null Values
Reviewed missing values and decided how to deal with them.

Removed Unnecessary Data
Dropped columns and rows that were not needed for the analysis.
